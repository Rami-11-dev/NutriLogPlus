# NutriLog+

**NutriLog+** - это Android-приложение для ведения журнала питания, отслеживания приёмов пищи и анализа качества питания.
Проект разрабатывается как портфолио-приложение с акцентом на Kotlin, Jetpack Compose, локальное хранение данных, работу с API и чистую архитектуру.

## Идея проекта
### Приложение помогает пользователю:
+ вести дневник питания,
+ добавлять продукты и блюда в приёмы пищи,
+ сохранять собственные рецепты,
+ отслеживать калории и макронутриенты,
+ получать информацию о продуктах из локальной базы и внешних источников.
+ Основная цель проекта - создать удобный мобильный инструмент для контроля питания и одновременно продемонстрировать навыки современной Android-разработки.

### Основные возможности
+ Ведение журнала питания по дням
+ Добавление приёмов пищи и отдельных продуктов
+ Работа с рецептами
+ Расчёт калорий, белков, жиров и углеводов
+ Локальная база продуктов на основе USDA FoodData SR Legacy
+ Получение информации о готовых продуктах через OpenFoodFacts API
+ Поиск продуктов по названию и штрихкоду
+ Хранение данных пользователя и прогресса
+ Подсчёт персональных норм питания и воды
+ Glassmorphism UI с поддержкой blur/haze-эффектов
